# Jobby App

A responsive job search platform built using React.js that allows users to browse jobs, filter results, search jobs, and view detailed job information.

## Features

- User Authentication (Login & Logout)
- Protected Routes
- Profile Details Section
- Search Jobs Functionality
- Employment Type Filters
- Salary Range Filters
- Location Filters
- Job Details Page
- Similar Jobs Section
- Loading View
- Failure View with Retry Functionality
- Responsive Design

## Tech Stack

### Frontend
- React.js
- JavaScript (ES6+)
- JSX
- CSS3

### Routing
- React Router DOM

### Authentication
- JWT Token
- Cookies

### API
- REST API Integration

## Installation

1. Navigate to project folder

```bash
cd jobby-app
```

2. Install dependencies

```bash
npm install
```

3. Start the application

```bash
npm start
```
